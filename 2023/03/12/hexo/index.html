2.安装

安装成功

hexo g = hexo generate
hexo s = hexo server
hexo d = hexo deploy

访问：http://localhost:4000

发布

新建 GitHub 仓库

我的仓库地址：https://github.com/lflwork
我的用户名：lflwork
我的域名：www.biji.work

新的仓库命名：用户名.github.io，设置公开权限
新的仓库名为：lflwork.github.io

添加密钥到 GitHub

生成密钥（已有密钥跳过）

添加密钥

配置网站信息

修改 _config.yml 文件以下选项

title: 工作笔记
subtitle: 学习每一步步步都算数
description: 工作学习笔记
keywords: java,php,app,软件,开发,产品,服务器,小程序
author: lflwork
language: zh-CN
timezone: Asia/Shanghai
url: https://www.biji.work
# 新增 root
root: /
new_post_name: :year-:month-:day-:title.md
deploy:
type: git
repo: git@github.com:lflwork/lflwork.github.io.git

repo 是 仓库的 ssh 地址

新增 CNAME 文件

在 source 目录下新增 CNAME 文件，内容为二级域名：www.biji.work
_posts markdown 源文件
images 图片目录

推送

每次推送都执行以下3条命令，避免缓存引起错误

推送成功

访问：lflwork.github.io 和 http://localhost:4000 页面一致
